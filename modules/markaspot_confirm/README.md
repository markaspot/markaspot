# Mark-a-Spot Confirm

This module provides confirmation functionality for service requests in Mark-a-Spot.

## Features

- Generates unique confirmation links for service requests
- Validates confirmation tokens
- Updates service request status upon confirmation
- Configurable confirmation workflows

## Integration

This module works with:

- Mark-a-Spot service request workflow
- Email notifications
- Status management

## Technical Notes

- Uses unique tokens for secure confirmation
- Implements routes for confirmation handling
- Provides service for generating confirmation links
