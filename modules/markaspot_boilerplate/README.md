# Mark-a-Spot Boilerplate

This module provides reusable text templates for service request responses in Mark-a-Spot.

## Features

- Predefined response templates for service requests
- Category-specific boilerplate text
- Customizable templates for different status transitions
- Integration with service request workflow

## Configuration

Boilerplate texts can be created and managed through the content management interface as Boilerplate nodes.

## Integration

This module works with:

- Mark-a-Spot service request workflow
- Status management
- Response handling
- Admin interfaces

## Technical Notes

- Uses Drupal's node system for template management
- Provides JavaScript integration for template selection
- Implements entity references for category-specific templates
