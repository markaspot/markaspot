# Mark-a-Spot Language

This module provides multilingual support for Mark-a-Spot installations.

## Features

- Language detection and switching
- Content translation for service requests
- Interface translation management
- Multilingual configuration options

## Integration

This module works with:

- Drupal's language system
- Content translation
- Interface translation
- Mark-a-Spot forms and displays

## Technical Notes

- Extends Drupal's language negotiation
- Provides custom services for language handling
- Implements configuration for translatable fields
- Enhances the multilingual user experience
